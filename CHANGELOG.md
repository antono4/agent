All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-01

### Added

- Initial release of CodeReviewBot
- AI-powered code review using GPT-4
- GitHub App integration with webhook support
- Automatic PR review with inline comments
- Support for multiple programming languages
- Docker and docker-compose deployment
- Comprehensive documentation
- GitHub Actions CI/CD workflows
- ESLint and Prettier configuration
- Unit and integration tests

### Features

- ✨ Intelligent code analysis
- 🔍 Detects bugs, security issues, performance problems
- 📝 Provides actionable suggestions
- 🎯 Supports 6+ programming languages
- 🚀 Production-ready deployment
- 📊 Quality score calculation
- 🔐 HMAC webhook validation
